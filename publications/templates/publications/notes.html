Notes

Study ID

If the data points from one publication come from different studies of the same intervention and the same outcome, then you should give them different "study ID" numbers, if they are statistically independent (not pseudoreplicates). However, if they come from the same study and they are not independent (pseudoreplicates), then you should either leave this blank or else you should give the same "study ID" number to all of the data points from the same study (if there are multiple data points from multiple studies in one publication). It will be possible to use random effects to model the non-independence of data points from the same publication and the same study (e.g., "random = ~ 1 | publication_ID/study_ID").

Study name

If you enter a "study ID" number, you can also enter a name for this "study ID" (optional).
